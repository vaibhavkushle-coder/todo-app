# 📝 Todo App

A simple and powerful Todo App built using React.

## 🚀 Features
- Add, Edit, Delete tasks
- Mark tasks as Done
- Filter (All / Done / Pending)
- Search tasks
- LocalStorage (data saved after refresh)
- Responsive UI
